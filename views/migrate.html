This WDC has moved to a new URL. Please update your connections:

New URL: https://json-xml-wdc.onrender.com

To update an existing Tableau data source:

  1. Open the workbook in Tableau Desktop.
  2. Go to Data → [your data source] → Edit Connection.
  3. Replace the URL with https://json-xml-wdc.onrender.com and reconnect.

If you use this WDC on Tableau Server, update your safelist command:

tsm data-access web-data-connectors add --name "JSON XML WDC" --url https://json-xml-wdc.onrender.com:443
tsm pending-changes apply
